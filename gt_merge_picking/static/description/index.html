Merge multiple pickings for large number of orders.

When we have large number of orders to process its difficult to open each order and process them, so this module will you know merge the orders for same customers and create new picking to reduce the time and work for the resources.

From tree view we can select more than one orders to merge the picking with same customers, and in action there is Merge picking which will open wizard to merge it.

After clicking on action merge picking wizard will open as shown in screenshot.

Once clicked on Merge new picking is created with different number and in source can be tracked which pickings are merged.

Trying to merge with different customers but it will not allow to merge and give you warning.



Trying to merge with different picking status will also not allow you.



If one picking is selected and try merge it will not allow.



Now will merge with Done status it will not allow to merge



Merging with two different picking type will not allow.
